- 3_Figure1_chromatin_compartments.ipynb
- 4_Figure1_DLR.ipynb
- fithic.ipynb
- Figure4_polycomb_interactions.ipynb
- Figure4_H3K9me3_profile.ipynb
- Figure5_snm3C-seq.ipynb
